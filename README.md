# YOLO Agent Skills

Public collection of practical agent skills, prompts, and workflows for Codex, Claude Code, OpenClaw, and related AI engineering agents.

The goal is to share reusable operational know-how without exposing private client data, credentials, internal infrastructure details, or proprietary business material.

## What Belongs Here

- Generic agent skills and workflows
- Cleaned-up prompts that are useful outside one private project
- SEO/AEO, deployment, debugging, review, and automation playbooks
- Templates that others can adapt safely

## What Does Not Belong Here

- API keys, tokens, secrets, cookies, or credentials
- Private customer data or lead lists
- Internal server IPs, hidden endpoints, or private repo paths
- Exact production configs that reveal sensitive architecture
- Content copied from private chats without sanitization

## Structure

```text
skills/        Reusable skill folders or markdown playbooks
examples/      Optional sanitized examples
```

## Publishing Rule

Before publishing anything, remove project-specific secrets and replace private details with placeholders such as `example.com`, `YOUR_TOKEN`, or `your-project-name`.
